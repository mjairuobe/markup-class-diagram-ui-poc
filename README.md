# Markup Mermaid Klassendiagramm

Dieses Repository baut ein Mermais-Klassendiagramm, das Markups wie Links und Highlighting enthält.

## Projektziel
Das Projekt war es Markups wie Hyperlinks und Highlighting-Marker in ein Klassendiagramm einzufügen, um damit Code-Änderungen einer Software darzustellen 
Damit soll die grafische Darstellung von Code-Änderungen ermöglicht werden, um Code Reviews mobile-first fähig zu machen.
